System Maintenance

📦 Data Backup

Download a snapshot of the entire system database, including all users, events, registrations, and photos. Use this for off-site storage or before making major changes.

🔄 System Restore

Restore the system to a previous state by uploading a backup file. This is useful for recovering from accidental data loss.

CAUTION: Restoring will PERMANENTLY DELETE all current data in the database and replace it with the contents of the backup file. This action cannot be undone.
